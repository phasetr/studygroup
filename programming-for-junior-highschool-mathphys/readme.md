課題も毎回作って適当に解説もしている.

## ipynb to md

必要なら次のコマンドで markdown 化.
ブログにあげるときは面倒なので画像はアップしていない.

```sh
# https://github.com/jupyter/nbconvert
$ jupyter nbconvert --to <output format> <input notebook>
$ jupyter nbconvert --to markdown 2020-.*.ipynb
```
